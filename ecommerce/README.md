# StepUp - E-commerce Sepatu Responsif

**StepUp** adalah aplikasi e-commerce sepatu berbasis **HTML, CSS, dan JavaScript murni (Vanilla)** tanpa framework. Didukung fitur lengkap dan tampilan **mobile-first** yang responsif di semua perangkat (HP, tablet, desktop).

## ✨ Fitur Lengkap

| Fitur | Keterangan |
|-------|-------------|
| 🔍 **Search & Filter** | Cari produk berdasarkan nama, kategori, ukuran, atau rentang harga |
| ❤️ **Wishlist** | Simpan produk favorit |
| 🛒 **Keranjang** | Kelola belanjaan real-time (tambah, kurangi, hapus item) |
| 📝 **Form Pesanan** | Isi alamat, & metode pembayaran |
| 📜 **Riwayat Pesanan** | Lihat semua pesanan yang sudah selesai/dibatalkan |
| ⭐ **Rating** | Beri rating bintang (1-5) untuk produk yang sudah dibeli |
| 📱 **Responsif HP** | Bottom navigation, tombol besar, layout 2 kolom untuk layar kecil |

## 🖥️ Teknologi

- **HTML5** - Struktur halaman
- **CSS3** - Styling & animasi (Flexbox, Grid, Media Queries)
- **JavaScript (ES6+)** - Logika interaktif & penyimpanan data (LocalStorage)
